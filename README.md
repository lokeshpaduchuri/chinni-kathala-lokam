# చిన్ని కథల లోకం

తెలుగు కథలు చిన్న హృదయాల కోసం — a mobile-first Angular storytelling library.

## Development

```bash
npm start
```

## Production build

```bash
npm run build
```

## Future enhancement

Make this an offline-capable PWA for bedtime reading. The current manifest, mobile metadata, safe-area layout, and icon placeholder prepare the app for a future Angular service-worker integration.
